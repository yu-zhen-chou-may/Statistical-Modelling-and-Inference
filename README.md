# Statistical Modelling and Inference

Coursework applying core biostatistics and causal inference methods in R, relevant to pharmacoepidemiology and real-world evidence analysis.

### Contents

**01 Stat** — Foundational statistical inference

**02 Logistic Regression Model** — Logistic regression for binary outcomes

**03 Linear Mixed Effects Model** — Linear mixed-effects models for repeated / clustered data

**04 Causal Analysis** — Causal inference methods

Each `.R` script is paired with a `.pdf` report of the analysis and results.
